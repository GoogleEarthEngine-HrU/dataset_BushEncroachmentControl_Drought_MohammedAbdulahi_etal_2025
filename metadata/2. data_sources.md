# Data Sources

This study used multiple satellite, reanalysis, and climate projection datasets.

- **Soil Moisture (SSMI)**  
  - Dataset: FLDAS (NOAH Land Surface Model)  
  - Resolution: 0.1°  
  - Period: 1982–2014  
  - Source: NASA GES DISC  
  - Purpose: Agricultural drought detection  

- **Runoff (SRI)**  
  - Dataset: ERA5-Land Reanalysis  
  - Resolution: 0.1°  
  - Period: 1982–2014  
  - Source: ECMWF Copernicus CDS  
  - Purpose: Hydrological drought detection  

- **Precipitation**  
  - Dataset: CHIRPS (Climate Hazards Center)  
  - Resolution: 0.05°  
  - Period: 1982–2014  
  - Purpose: Forcing, validation  

- **Temperature**  
  - Dataset: CHIRTS (Climate Hazards Center)  
  - Resolution: 0.05°  
  - Period: 1982–2014  
  - Purpose: Forcing, validation  

- **Land Use / Land Cover (Shrub Encroachment)**  
  - Dataset: Dynamic World V1.6.1 (Google/WRI)  
  - Resolution: 10 m (aggregated to 0.1°)  
  - Period: 2015–2024  
  - Purpose: Shrubland extent, scenario development  

- **Future Climate Scenarios**  
  - Dataset: NEX-GDDP CMIP6 downscaled projections  
  - Resolution: 0.25° (upscaled to 0.1°)  
  - Period: 2015–2100  
  - Source: NASA NCCS  
  - Scenarios: SSP2-4.5 (moderate), SSP5-8.5 (high emissions)  
  - Purpose: Future drought projection  

- **Drought Event Identification**  
  - Method: Run theory (Yevjevich, 1967)  
  - Thresholds: SSMI < -1.0, SRI < -0.5  
  - Metrics: Duration, Severity, Intensity
